## 🚀 Setup and Installation

1. Clone the repository
   ```bash
   git clone https://github.com/yourusername/your-portfolio.git
   ```

2. Navigate to the project directory
   ```bash
   cd your-portfolio
   ```

3. Install dependencies
   ```bash
   npm install
   ```

4. Start the development server
   ```bash
   npm run dev
   ```
